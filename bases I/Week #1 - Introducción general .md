# Comparación de motores de bases de datos y sus arquitecturas

Se van a seleccionar algunos motores de bases de datos y vamos a crer una matrix comparativa entre ellos, las columnas son los motores de bases de datos y las filas son las características. Para cada característica extraiga para que funciona y las virtudes algoritimicas de la implementación del fabricante. 

Motores relacionales:
- PostgreSQL
- SQL Server
- MariaDB
- MySQL

Motores documentales:
- MongoDB
- CosmosDB
- DynamoDB
- Redis

Categorías para averiguar características clave:
- Modelado de datos 
- Seguridad
- Concurrencia
- Recuperación ante fallas
- Rendimiento y Optimización
- Lenguajes para manipular la base de datos
- Sistemas Operativo

2. Averiguamos características "wow" por cada categoría, unas 2 características por categoría, usamos la misma características para todos los motores.

3. Generemos la matriz comparativa. 

4. Deduzcamos diferencias, que cosas tienen negativas o positivas comparativamente, cuál de los motores cree usted qué es mejor?
